# actions

- `navigation.rs`：Tree、Index Map、Codex Runs 和右侧预览的上下/底部/重置移动。
- `focus.rs`：左右焦点切换与当前导航区焦点解析。
- `tree.rs`：tree 展开/折叠、reveal path 与选中项刷新。
- `display.rs`：搜索、帮助、行号与文字缩放开关。
- `index_map.rs`：`index.md` 地图预览、跳转与 tree reveal 动作。
- `nav_mode.rs`：`II` 双击切换 Tree 与 Index Map。
- `preview.rs`：全屏文件预览的打开、关闭、滚动与刷新动作。
- `tests.rs` / `tests/`：pad sider 动作层 tree、Index Map、预览、显示与 tick 回归测试。
