# title_summary

- `types.rs`：标题摘要请求结果、wire API 类型与默认模型常量。
- `window.rs`：是否启用、刷新阈值与 turn 窗口选择。
- `normalize.rs`：模型返回标题的前缀、包裹符与长度清理。
- `prompt.rs`：标题摘要 prompt 构造。
- `request.rs`：Responses/Chat 请求发送与重试入口。
- `response.rs`：Responses/Chat 响应和错误文本解析。
- `util.rs`：日志截断等共享小工具。
- `tests.rs`：标题摘要窗口、wire API 与标题清理回归测试。
