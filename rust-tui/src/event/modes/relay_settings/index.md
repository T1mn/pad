# relay_settings

- `mod.rs`：relay 设置模式入口。
- `actions.rs` / `actions/`：agent/provider/detail 三个区域的键盘动作分发。
- `edit.rs` / `edit/`：字段编辑、provider 增删与 relay 配置持久化。
- `popup.rs` / `popup/`：弹层入口、OpenCode models 编辑和模型选择器。
- `transfer.rs`：relay 配置迁移/传输。
- `tests.rs` / `tests/`：relay settings 键盘流、provider/model 编辑与持久化回归。
